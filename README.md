Purpose: The 'Fylde Aero Inventory Managment System' (File name: app.py) is a command-line based program developed in Python. The application is used to Store, Add, Delete and View Items within the the 'inventory.csv' CSV file. The CSV file is simple and protected by input validation, controlling data type entry.
Software Requirements: Python 3.13, an appropriate runtime environment 
How to  run: 
1. Clone Repository
2. run python app.py 
3. At Runtime, you will be presented with the inventory main menu. 1) Add Item, 2) Delete Item, 3) list Items 4) Quit
4. Input the number of the action you would like to perform
5. Example output for 1) Add Item:
   
   ***Add a New Item to Inventory***
   Enter Item Name: Hammer
   Enter Item Quantity (Whole Integer): 100
   ***Item Successfully added to Fylde Aero Inventory Management System***
